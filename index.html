HarshScan is a Python-powered content analysis engine that scans large text datasets for harmful & harsh language, computes sentiment scores (positive / negative / neutral), and generates detailed chunk-by-chunk reports — all processed in parallel across multiple CPU cores for maximum throughput.

Upload .txt, .csv, or .xlsx files (even multiple at once), and get instant visualizations, a searchable results table, severity scoring on a 1–10 scale, and downloadable CSV + SQLite reports — no external dependencies beyond Python's standard library and Flask.

Python 3.8+ Flask API Parallel Processing Sentiment Analysis Harsh Word Detection SQLite Storage CSV Export Multi-file Support 50K+ Records ThreadPoolExecutor
